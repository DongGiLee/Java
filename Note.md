    - class
        > 보통 클래스는 클래스명.java 파일에 단독으로 작성하는것이 일반적이다.
        ex)_ class Animal{...} => Animal.java

        - 객체 변수는 공유되지않는다.
    - method
        > 타언어는 함수라는 것이 별도로 존재한다. 하지만 자바는 클래스를 떠나 존재하는 것은 있을수 없기 때문에 자바의 함수는 따로 존재하지않고 클래스 내에 존재한다.

        > 자바는 이러한 클래스 내의 함수를 메서드라고 부른다.
